# Citation Standards

This document defines the "Threshold of Truth" for public documentation on `garrettmanley.dev`.

## Tier 1: Canonical (Automatic Authority)
- **RFCs**: Official Internet protocols.
- **Official Documentation**: `learn.microsoft.com`, `firebase.google.com`, `modelcontextprotocol.io`.
- **Peer-Reviewed Papers**: Found via Semantic Scholar or arXiv.

## Tier 2: Expert (Requires Corroboration)
- **Primary Research Blogs**: Netflix Tech Blog, Uber Engineering, AWS Architecture.
- **Standards Organizations**: W3C, IEEE.

## Tier 3: Prohibited for Documentation
- Personal blogs (unless from a core maintainer).
- Reddit/Stack Overflow (use for leads only).
- Medium/Dev.to (use for leads only).

## Documenting Evidence
Every "Fact" in a spec must follow this format:
> "Fact statement here" [1]
> 
> **Proof**: `cmd output snippet`
> **Citation**: [1] URL to authoritative source.
